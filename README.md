# Aza text encryptor

## General info
Aza this is a text encryption software
Help command:
```python aza.py --help```

## Examples
```
python aza.py --text "Sample text here" --method base64 --tofile 0
python aza.py --text "Sample text here" --method binary --tofile 0
```
	
## Setup
To run this software you need:

* Install python
* Open setup.bat
